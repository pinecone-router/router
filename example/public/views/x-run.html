• You can use the x-run attribute to conditionally run scripts when a route is loaded.
  • x-run.once will ensure the script only runs once, even if the route is revisited.
  • x-run:on="expression" will run the script only when the expression evaluates to true.
  • You can also combine both modifiers like so: x-run.once:on="expression"

    • Check the console to see the results.
